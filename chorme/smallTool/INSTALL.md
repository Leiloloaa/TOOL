# 安装指南

## 快速开始

### 1. 准备图标文件（可选）

如果没有图标文件，插件仍然可以正常工作，但会显示 Chrome 的默认图标。

如果需要添加图标：
1. 准备三个尺寸的 PNG 图标：
   - 16x16 像素 → `icons/icon16.png`
   - 48x48 像素 → `icons/icon48.png`
   - 128x128 像素 → `icons/icon128.png`

2. 将图标文件放入 `icons/` 目录

### 2. 加载扩展

1. 打开 Chrome 浏览器
2. 在地址栏输入 `chrome://extensions/` 并回车
3. 开启右上角的 **"开发者模式"** 开关
4. 点击左上角的 **"加载已解压的扩展程序"** 按钮
5. 选择本项目的 `chorme/smallTool` 目录
6. 扩展安装完成！

### 3. 使用扩展

1. 点击浏览器工具栏中的扩展图标（如果没有看到，点击拼图图标查找）
2. 在弹出的窗口中：
   - 在"输入中文"文本框中输入要翻译的中文
   - 点击"翻译"按钮，或使用快捷键 `Ctrl/Cmd + Enter`
   - 翻译结果会显示在"英文翻译"文本框中
3. 点击"清空"按钮可以清除所有内容

## 故障排除

### 扩展无法加载

- 确保选择了正确的目录（`chorme/smallTool`）
- 检查 `manifest.json` 文件是否存在且格式正确
- 查看扩展管理页面的错误信息

### 翻译功能不工作

- 检查网络连接
- 打开浏览器控制台（F12）查看错误信息
- 某些网络环境可能需要代理才能访问翻译 API

### 图标不显示

- 图标文件是可选的，不影响功能
- 如果需要图标，确保文件路径和尺寸正确
- 图标文件必须是 PNG 格式

## 更新扩展

1. 修改代码后，在 `chrome://extensions/` 页面
2. 找到"小工具集合"扩展
3. 点击刷新图标 🔄
4. 扩展会自动重新加载
