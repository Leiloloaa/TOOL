# 标签页 - Chrome 新标签页插件

一个美观、功能丰富的 Chrome 起始页插件，支持组件模式和极简模式，动态壁纸，多搜索引擎切换。

## ✨ 功能特性

### 🎨 两种显示模式

- **极简模式**：只显示时间、日期和搜索框，干净清爽
- **组件模式**：在右侧显示日历和倒计时组件，功能丰富

### 🖼️ 动态壁纸

- 支持**视频动态壁纸**和**静态图片壁纸**
- 内置多种精美壁纸，来自高质量免费图库
- 可随时在设置中切换

### 🔍 多搜索引擎支持

- Google（默认）
- Bing
- 百度
- DuckDuckGo
- 搜索框居中偏上，输入即搜索

### ⏰ 时间与日期

- 大字体实时时钟显示
- 完整日期和星期显示
- 秒级更新

### 📝 随机诗句

- 底部随机显示中国古诗词名句
- 包含 100+ 首经典诗词
- 每 30 秒自动更换

### 📅 日历组件

- 完整月历视图
- 高亮今日日期
- 周末日期特殊标记
- 支持月份切换

### ⏳ 倒计时组件

- 自定义倒计时事件
- 显示天、时、分、秒
- 翻页动画效果
- 到期提醒

## 📦 安装方法

### 开发者模式安装

1. 下载或克隆本项目到本地

2. 打开 Chrome 浏览器，进入扩展程序页面
   - 在地址栏输入：`chrome://extensions/`
   - 或通过菜单：更多工具 → 扩展程序

3. 开启右上角的「开发者模式」

4. 点击「加载已解压的扩展程序」

5. 选择 `startPage` 文件夹

6. 安装完成！打开新标签页即可体验

## 🎯 使用指南

### 基本操作

- **搜索**：直接在搜索框输入内容，按回车或点击搜索按钮
- **切换搜索引擎**：点击搜索框左侧的搜索引擎图标
- **打开设置**：点击右下角的齿轮图标

### 设置选项

- **显示模式**：在极简模式和组件模式之间切换
- **壁纸类型**：选择动态视频或静态图片
- **选择壁纸**：从预设壁纸中选择喜欢的
- **组件开关**：显示或隐藏日历/倒计时组件
- **倒计时设置**：设置倒计时的事件名称和目标时间

## 🔧 自定义开发

### 添加新壁纸

编辑 `data/wallpapers.js`，在对应数组中添加新壁纸：

```javascript
{
  id: 'v10',
  name: '壁纸名称',
  thumbnail: '缩略图URL',
  url: '壁纸URL'
}
```

### 添加新诗句

编辑 `data/poems.js`，在 `POEMS` 数组中添加：

```javascript
"诗句内容 —— 作者《诗名》"
```

### 添加新搜索引擎

编辑 `newtab.js` 中的 `SEARCH_ENGINES` 对象，并在 `newtab.html` 中添加对应的选项。

## 📋 注意事项

1. 壁纸资源来自网络，首次加载可能需要一些时间
2. 动态壁纸可能会消耗更多系统资源，如有性能问题可切换到静态壁纸
3. 所有设置都保存在本地，不会同步到其他设备
