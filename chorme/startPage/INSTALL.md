# 安装指南

## 快速安装步骤

### 1. 打开 Chrome 扩展管理页面

在 Chrome 地址栏输入：
```
chrome://extensions/
```

### 2. 开启开发者模式

点击页面右上角的「开发者模式」开关

### 3. 加载扩展

点击「加载已解压的扩展程序」按钮，选择 `startPage` 文件夹

### 4. 完成！

打开新标签页即可看到效果

---

## 生成图标（可选）

如果需要自定义图标：

1. 用浏览器打开 `icons/generate-icons.html`
2. 下载生成的图标文件
3. 将图标文件保存到 `icons/` 文件夹
4. 取消 `manifest.json` 中图标配置的注释
5. 重新加载扩展

---

## 常见问题

### Q: 壁纸加载不出来？
A: 首次加载壁纸需要联网，请确保网络正常。壁纸来自 Unsplash 和 Mixkit。

### Q: 如何恢复默认设置？
A: 在设置面板中可以重置所有设置，或者清除浏览器的本地存储数据。

### Q: 设置不保存？
A: 请确保授予了 storage 权限，可以在扩展详情页查看。

---

## 本地测试

如果只想本地预览效果（不安装到 Chrome），可以直接用浏览器打开 `newtab.html` 文件。

注意：本地测试时部分功能（如设置保存）会使用 localStorage 而非 Chrome Storage API。
